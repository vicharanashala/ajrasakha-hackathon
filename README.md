# Static Website for Hackathon

This repository contains a static website built for a hackathon using Node.js and npm.

## Instructions

1. Install Node.js (v16 or above recommended)
2. Clone the repository
3. Install dependencies using:
   npm install
4. Start the project locally using:
   npm run start
5. Open your browser and visit:
   http://localhost:3000
6. To push changes to the repository:
   git add .
   git commit -m "your commit message"
   git push

The application runs on port 3000 by default.
